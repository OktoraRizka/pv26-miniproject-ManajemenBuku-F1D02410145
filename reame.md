Sistem Manajemen Buku Perpustakaan

Sistem Manajemen Perpustakaan adalah aplikasi desktop bebasis GUI yang bertujuan untuk mempermudah pengelolaan data buku, data anggota dan transaksi peminjaman secara efisien.

Aplikasi ini memiliki sistem Role-Based Access Control, yang dimana terdapat 2 user yaitu : Super admin, dan Peminjam   


Aplikasi ini dibuat menggunakan :
- Bahasa : Python
- Framework : PySide6
- Database : SQLite3
- Styling : QSS
